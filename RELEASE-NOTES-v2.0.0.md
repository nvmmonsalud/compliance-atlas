# v2.0.0 — Audio-first release 🌊

## What changed

The first public release shipped a dashboard. v2.0 ships a **briefing**.

Every morning at 06:00 UTC, Compliance Atlas now generates a 30-second audio rundown of the day's top regulatory signals. Press play on the morning coffee, get caught up in 30 seconds, dig into a specific region if you want the full picture.

## What's new

- **Daily audio briefing** at the top of the dashboard
  - 30s TTS rundown of today's top signals
  - Live timestamp + critical/high count overlay
  - One-tap playback, native browser audio controls
- **Per-jurisdiction deep dives**
  - 5 audio briefs, one for each region: US Federal, EU, UK, Singapore, Japan
  - Jurisdiction cards with custom SVG cover art (Capitol, EU stars, Big Ben, Marina Bay, Mt Fuji)
  - Per-card play/pause with glow effect
  - Live finding counts pulled from the API
- **New 1200×630 OG image** with v2.0 branding + jurisdiction network map
- **Updated Reel kit** (`public/REEL.md`)
  - v2.0 launch narrative emphasizing audio-first
  - 6 ready-to-ship TTS scripts (daily + 5 jurisdictions) — reuse as podcast ads, Telegram voice memos, Twitter Spaces intros
  - Updated 60s/90s cuts, 4-platform caption pack, storyboard
- **v2.0 version pill** in the hero

## What stayed the same

- False-failure reveal: 5,490 → 190, 28 → 5, 24/7 → real-time
- Live dashboard with KPIs, top critical findings, cross-jurisdiction conflicts
- Methodology, GitHub CTA, footer
- API: all endpoints CORS-open, no auth, no rate limit
- Data: same shape, same filters, same 5 jurisdictions

## Try it

```bash
git clone https://github.com/nvmmonsalud/compliance-atlas
cd compliance-atlas
git checkout v2.0-fullstack
pip install -r requirements.txt
python curate.py
gunicorn api_combined:app --bind 0.0.0.0:8080
# Open http://localhost:8080
# Press play on the daily briefing
# Click any jurisdiction card for a region-specific brief
```

## API (unchanged)

| Endpoint | Returns |
|---|---|
| `GET /api/summary` | Aggregate stats + jurisdiction counts (now powering the briefing meta + jurisdiction card counts) |
| `GET /api/findings` | All findings (with `?severity`, `?jurisdiction`, `?sector`, `?limit` filters) |
| `GET /api/conflicts` | Cross-jurisdiction conflict cards |
| `GET /api/arbitrage` | Operational arbitrage opportunities |
| `GET /api/sectors` | Sector breakdown |
| `GET /api/health` | Liveness check |
| `GET /audio/daily-briefing.mp3` | **NEW** — 30s daily audio briefing |
| `GET /audio/jur-{us-federal,eu,uk,sg,japan}.mp3` | **NEW** — 5 per-jurisdiction audio briefs |

## Behind the build

This release was shipped using the full Hermes agent stack:

- **TTS**: 6 audio files generated via `text_to_speech` (edge-tts provider, free)
- **Image gen**: pivoted to SVG cover art when FAL was unavailable — turned out cleaner & more brand-consistent anyway
- **OG image**: PIL-generated 1200×630 with jurisdiction network map
- **Vision**: verified site + OG image visually via `browser_vision` and `vision_analyze`
- **Delegate task**: regswarm pipeline still running (5 jurisdictions, 12 agents)
- **Browser**: live-tested the v2.0 site end-to-end before release
