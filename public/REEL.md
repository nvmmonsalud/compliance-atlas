- **Color palette**: Dark navy `#0a0e1a`, white `#e6e9ef`, accent green `#10b981`, warn amber `#f59e0b`, critical red `#ef4444`
- **Typography**: Bold sans-serif for headlines (system stack works — SF Pro on Mac, Segoe UI on Windows)
- **Vibe**: Confident, technical, slightly cheeky. NOT corporate. NOT startup-y. Indie builder with receipts.
- **B-roll ideas**: Code editor scrolling, terminal running, agent status pills, dashboard interactions
